Dossiê Ergonómico App
Este projeto é uma aplicação desktop desenvolvida para a gestão e análise de dados ergonómicos, permitindo a identificação de variáveis críticas (como CO2 e calor) e o seu impacto na produtividade industrial. O sistema foi concebido para transformar dados brutos em relatórios estruturados e decisões baseadas em evidências.

Funcionalidades
Login de Avaliador: Acesso seguro para processamento de dados.

Gestão de Questionários: Interface dinâmica para inserção e coleta de dados ergonómicos (utilizando metodologias como NASA-TLX/COPSOQ).

Dashboard Analítico: Visualização clara das métricas coletadas.

Customização de Relatórios: Exportação e formatação de resultados para análise técnica.

Tecnologias Utilizadas
React: Construção da interface do utilizador.

TypeScript: Garantia de tipagem e segurança no código.

Electron: Empacotamento da aplicação para ambiente Desktop (Windows).

Vite: Build tool rápida para o desenvolvimento.

Como Rodar o Projeto
Pré-requisitos:
Certifica-te de que tens o Node.js instalado na tua máquina.

Clonar o repositório:

Bash
git clone https://github.com/HCE-DevEng/dossie-ergonomico-app.git
cd dossie-ergonomico-app
Instalar dependências:

Bash
npm install
Iniciar o modo de desenvolvimento:

Bash
npm run dev
